HTMLパス変換ツール

このツールについて

GitHub Pagesなどで使用するHTMLファイルのパスを、ローカル環境で動作するように変換します。

変換対象の属性：

  • href, src - 通常のリンクや画像
  • data-src - 遅延読み込み（lazy loading）
  • poster - videoタグのサムネイル画像
  • background - 背景画像
  • action - フォームの送信先URL
  • srcset - レスポンシブ画像
  • content - metaタグのURL
  • manifest - PWAのマニフェストファイル
  • style内のurl() - CSS背景画像

HTMLファイルをここにドラッグ＆ドロップするか、クリックしてファイルを選択してください
